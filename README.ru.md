# Автотесты — SauceDemo

## Описание проекта

Данный проект содержит автоматизированные UI-тесты для веб-приложения SauceDemo.

Цель проекта — продемонстрировать полный QA-процесс:
- Планирование тестирования
- Дизайн тестов (чек-листы и тест-кейсы)
- Автоматизация UI
- Отчётность и CI

Тестируемое приложение: https://www.saucedemo.com/

---

## Область тестирования

Покрыты следующие сценарии:

- Авторизация пользователя
- Каталог товаров
- Работа с корзиной
- Процесс оформления заказа

## ▶️ Запуск тестов

1. Установка зависимостей:
```bash
pip install -r requirements.txt
```

2. Запуск тестов:
```bash
pytest
```

3. Запуск с Allure:
```bash
pytest --alluredir=reports
allure serve reports
```

---

## Отчётность

Результаты тестов формируются с помощью Allure Report и включают:
- Шаги выполнения тестов
- Скриншоты при падении
- Логи

---

## CI интеграция

Тесты автоматически запускаются через GitHub Actions при каждом push.
